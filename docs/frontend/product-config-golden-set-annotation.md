# Golden Set 标注工作台

入口为 `/agent/golden-set`。任务页显示当前席位进度；标注页左栏为冻结、脱敏后的证据，右栏为中文表单。产品包允许多个平级产品及附件组件、销售套件、不确定关系；ERP 身份通过独立只读搜索选择 `Company + PartNum`，不展示系统候选。

页面自动保存草稿，保存并下一条会提交当前席位，离开页面会提示。标注页不展示 prediction、匹配状态、另一位标注员答案或内部标识。复核仅供管理员使用，且只出现双方均提交的任务。

试标清单为固定 baseline 的 package 20 与 ERP 30 分层子集；系统只创建任务，不自动填写人工答案，也不生成准确率。
